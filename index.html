Zero Emotions TikTok App

Demo веб-страница для TikTok for Developers. Показывает: авторизацию через Login Kit и загрузку видео в черновики через Content Posting API.

1. Login with TikTok

Нажмите кнопку, чтобы перейти на TikTok OAuth и авторизовать приложение.

2. Upload video (draft)

После авторизации пользователь выбирает файл видео и отправляет его на сервер для загрузки в TikTok как черновик.


В реальной интеграции этот интерфейс вызывает backend, который использует TikTok Content Posting API (video.upload) для отправки файла в черновики TikTok-аккаунта авторизованного пользователя.
